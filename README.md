# Desafio Texo It - Awards Interval
## API Endpoints
| METHOD | URI |
|------|------------------|
| GET | /movies |
| GET | /movies/{id} |
| POST | /movies |
| PUT | /movies/{id} |
| DELETE | /movies/{id} |
| GET | /moveis/awards/interval/ |
|------|------------------|

## Compile application

```
	mvn package
```

## Run application

```
	java -jar ./target/desafiotexoit-0.0.1.jar
```

## H2 Console
